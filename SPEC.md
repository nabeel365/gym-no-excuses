# Gym Website Specification

## Project Overview
- **Name**: NO EXCUSES Fitness
- **Type**: Premium Interactive Fitness Platform
- **Core Functionality**: High-energy gym website with personalization, transformation previews, AI coaching, gamification, and booking
- **Target Users**: Fitness enthusiasts, beginners to advanced athletes

## Tech Stack
- Next.js 14 (App Router)
- React 18
- TypeScript
- Tailwind CSS
- GSAP for animations
- Framer Motion
- Zustand for state management
- Lucide React for icons

## Color Palette
- **Primary**: #0a0a0a (Deep Black)
- **Secondary**: #1a1a1a (Dark Grey)
- **Accent 1**: #ff3b3b (Neon Red)
- **Accent 2**: #00ff88 (Neon Green)
- **Text Primary**: #ffffff
- **Text Secondary**: #a0a0a0

## Typography
- **Headings**: Bold, uppercase, strong
- **Body**: Clean, modern sans-serif

## Core Features
1. Cinematic Landing Page with video hero
2. Fitness Goal Quiz (5 questions)
3. Body Transformation Preview
4. AI Fitness Coach Chat
5. Gamified Dashboard
6. 3D Gym Tour
7. Trainers Section
8. Transformation Stories
9. Membership & Pricing
10. Class Booking System
11. Workout & Diet Generator
12. Workout Mode Toggle
